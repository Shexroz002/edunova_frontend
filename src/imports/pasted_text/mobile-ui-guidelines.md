Do not redesign the entire UI and do not change the visual style. Keep the same branding, colors, typography, spacing style, and design language from the desktop version. The task is to optimize the layout for mobile screens while maintaining the original design identity.

Main Goal

Convert the current desktop page into a clean, modern mobile web layout with proper responsive structure and mobile UX best practices.

Mobile Frame

Use a 390px width mobile frame (standard smartphone size).

Layout Rules

Convert the layout to a mobile-first vertical structure

Stack all sections vertically

Avoid horizontal scrolling

Maintain clear spacing between sections

Ensure all elements stay inside the screen width

Use Auto Layout for responsive behavior

Navigation

If the page has a desktop navbar, convert it into a mobile navigation pattern

Use either:

Hamburger menu, or

Compact top navigation

Remove unnecessary desktop navigation elements

Components Adaptation

Adjust desktop components to mobile-friendly structures:

Cards / Stats

Convert to single-column card layout

Keep cards readable and well spaced

Tables

Convert tables into stacked data cards or list blocks

Sidebars

Convert sidebar navigation into:

hidden drawer menu

collapsible navigation

or bottom navigation if appropriate

Buttons

Minimum 44px height

Full-width or easily tappable buttons

Clear primary vs secondary actions

Forms

Convert to single-column layout

Proper spacing between fields

Touch-friendly inputs

Charts / Data Blocks

Simplify large charts

Ensure they fit mobile width properly

Typography

Scale typography for mobile readability:

Clear heading hierarchy

Body text readable on small screens

Avoid overly large titles or cramped text

Spacing

Use consistent padding and margins

Maintain a clean SaaS-style layout

Avoid dense or cluttered UI

Visual Style

Preserve the original color palette

Maintain the same design language

Use modern mobile UI patterns

Clean shadows, rounded corners, and balanced spacing

Keep the interface minimal, professional, and polished

Output

Create a high-fidelity mobile web version of the page that:

Preserves the original design identity

Uses proper mobile UX

Is structured with Auto Layout and responsive constraints

Looks like a professional SaaS mobile web interface
